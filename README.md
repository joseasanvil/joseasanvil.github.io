# joseasanvil.github.io

Personal website for Jose A. Sanchez Viloria, published with GitHub Pages.

## Local Preview

Open `index.html` in a browser. No build step is required.

## Public Pages

- `index.html`: professional profile and portfolio
- `resume.html`: semantic, crawlable resume
- `Resume/Resume_Full.pdf`: downloadable resume

Search discovery is supported by `robots.txt`, `sitemap.xml`, canonical URLs, social metadata,
and Schema.org structured data.
